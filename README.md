ADA - Ambiente Digital de Aprendizagem

> Algoritmos e Programação I - Unichristus (2025.2)

Este projeto é um Ambiente Digital de Aprendizagem (ADA) desenvolvido como Produção Técnica para a monitoria da disciplina de Algoritmos I. O objetivo é fornecer um material de apoio estático, acessível e focado na autonomia do estudante.

Acesse o site oficial: [Clique aqui para ver o ADA online](https://TomazGalileu.github.io/ada-algoritmo/)

---

Objetivos
  Apoiar a aprendizagem de lógica de programação (C/C++).
  Promover a autonomia através de trilhas com gabaritos comentados.
  Sistematizar dúvidas frequentes dos plantões de monitoria.

Trilhas de Conteúdo
O ambiente está organizado em módulos progressivos:
1.  Variáveis e E/S: Tipos primitivos, `printf`, `scanf`.
2.  Seleção: Estruturas condicionais (`if`, `switch`).
3.  Repetição: Laços (`for`, `while`, `do-while`).
4.  Modularização: Criação e uso de funções.
5.  Vetores: Arrays unidimensionais.
6.  Avançado: Matrizes e Structs.

Tecnologias Utilizadas
  HTML5 & CSS3: Estrutura e estilo.
  Pico.css: Framework CSS minimalista para design limpo.
  Prism.js: Biblioteca para coloração de sintaxe de código C.
  GitHub Pages: Hospedagem estática e versionamento.
